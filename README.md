Bank Loan Analytics Dashboard (Excel)
This project contains a fully interactive Bank Loan Analytics Dashboard built in Microsoft Excel using advanced formulas, PivotTables, slicers, chart and UI formatting.
The dashboard provides insights into loan distribution, customer segments, transaction trends, verification status, and branch-wise performance.




 Project Overview
This Excel dashboard gives a complete 360° view of bank loan performance across:
Loan Amounts
Interest Income
Collection Trends
Customer Demographics
Loan Maturity
Verification Status
Transaction Analysis
Suspicious Activity & Risk Monitoring
It is designed for loan officers, bank managers, financial analysts, or anyone needing clear and actionable financial insights.
